# dreamer_bot
Discord bot

## Python Dependencies
- python3.6
- discord.py
- pyyaml
- systemd

## Usage
Create files called `auth.yaml` and `replies.yaml`.
Create a directory `reply_files/`

### auth.yaml
Add your discord bot token

```
token: 'YOUR_DISCORD_BOT_TOKEN'
```

### replies.yaml
```
{}
```

### Run the bot
`python bot.py`
